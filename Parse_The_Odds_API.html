Use this endpoint to access any available markets for an event. For the most popular featured markets, use the /v4/sports/{sport}/odds endpoint (see above). historical events

Historical events and bookmaker odds as they appeared at a specified point in time. Historical snapshots for featured markets are available from June 2020, at 10 minute intervals until Sep 2022, and at 5 minute intervals thereafter. Snapshots of historical odds for non-featured markets (player props, period markets, alternates, etc.) are available from May 2023 at 5 minute intervals.

Find out more

Returns bookmaker odds for a single event as they appeared at the specified timestamp (date parameter). Accepts any available market keys using the markets parameter. Use this endpoint to access historical odds from any available markets for an event. For historical odds of the most popular featured markets, use the /v4/historical/sports/{sport}/odds endpoint. Returns a list of participants for a given sport. Depending on the sport, a participant can be either a team or an individual. For example for NBA, this endpoint returns a list of teams. For tennis, it will return a list of players. Participants are used to define events, for example Boston Celtics vs Chicago Bulls, or Novak Djokovic vs Roger Federer. This endpoint does not return players on a team. The returning list should be considered a whitelist and may include participants that are no longer active. Future updates will remove inactive particpants.
