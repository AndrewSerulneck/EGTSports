## Key UI Features

### 1. Collapsible Section
- **Default State:** Collapsed (hidden)
- **Trigger:** Click on "ğŸ“Š Quarter & Halftime Odds (Optional)"
- **Benefit:** Keeps main interface clean, reveals only when needed

### 2. Sport-Specific Display
- **Shown for:** NFL, NBA, College Football, College Basketball, NHL, MLB
- **Hidden for:** Boxing, UFC, World Cup, MLS
- **Logic:** Conditional rendering based on sport type

### 3. Grid Layout
- **Spreads/Moneylines:** 2-column grid (Away | Home)
- **Totals:** Full-width single input
- **Responsive:** Adapts to mobile screens

### 4. Clear Labeling
- **Period Headers:** Bold, slightly smaller font (14px)
- **Input Placeholders:** Include team name and period (e.g., "Los Angeles Rams Q1 Spread")
- **Visual Separators:** Subtle borders between periods

### 5. Input Behavior
- **Real-time Updates:** Changes immediately update game state
- **Empty Values:** Allowed (fields optional)
- **Format Flexibility:** Accepts both "+3.5" and "3.5" (auto-formatted on save)

## Notes for Developers

1. **Optional Fields:** All quarter/halftime fields are optional - if The Odds API doesn't provide them, fields remain empty
2. **Validation:** No strict validation - admins can enter any value
3. **Format Preservation:** Values saved exactly as entered (formatting happens client-side)
4. **Performance:** Collapsible section prevents DOM bloat when not in use
5. **Backwards Compatible:** Existing code continues to work - quarter/halftime fields are additive
